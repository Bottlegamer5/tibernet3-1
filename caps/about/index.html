About Captrack

Captrack is a website that tracks the best Bottle Cap collections on record.

This website is static and does not use any server-side code and the website cannot provide any sort of personalised experience. A small amount of data about you is sent to Netlify which is the VPS used for this site but it is minimal and mostly information which is required for the site to work at all like your IP address. I do not get ANY data like that though, only Netlify. if you are worried for whatever reason, check with Netlify's documentation or Terms of Use for information. Google Fonts is also used on this site.
